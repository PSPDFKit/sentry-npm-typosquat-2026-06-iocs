# Security

This repository contains IOCs and detection tooling only. It intentionally does not contain malware samples.

Do not run:

```bash
⚠️ MALICIOUS: npx @⚠️sentry-internals/profiling-node --diagnose
⚠️ MALICIOUS: npx @⚠️sentry-browser-sdk/profiling-node --diagnose
```

If you need to handle package samples, do so only in an isolated malware-analysis environment.
